# Template Primitives (Canonical)

These primitives define the complete scope of what Phase 5 templates may express.
No template may exist outside these categories without explicit governance approval.

## Core Result Primitives
- ASSERTION_RESULT
- CONDITIONAL_RESULT
- NEGATED_RESULT
- SCOPE_LIMITATION

## Dependency Primitives
- DEPENDENCY_DISCLOSURE
- REQUIRED_PRECONDITION
- UNSATISFIED_DEPENDENCY

## Proof and Validation Primitives
- PROOF_CONFIRMED
- PROOF_FAILED
- VALIDATION_STATUS

## Constraint and Audit Primitives
- CONSTRAINT_DISCLOSURE
- AUDIT_CONFIRMATION
- ARTIFACT_REFERENCE

## Explanation Wrappers
- TECHNICAL_EXPLANATION
- DECLARATIVE_EXPLANATION
- PEDAGOGICAL_EXPLANATION

This list is intentionally minimal and closed.
