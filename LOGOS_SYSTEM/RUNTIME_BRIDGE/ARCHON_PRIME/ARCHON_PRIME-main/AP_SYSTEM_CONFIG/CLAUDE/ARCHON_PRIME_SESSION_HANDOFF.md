ARCHON PRIME SESSION HANDOFF
Unified Context Transfer Artifact

Purpose

This artifact governs how operational context is transferred between AI sessions
participating in the Archon_Prime system.

The artifact ensures:

• zero workflow loss
• deterministic session initialization
• preservation of architectural context
• preservation of governance rules
• preservation of runtime phase
• preservation of current operational objective

SECTION 1 — GOVERNANCE ACTIVATION

Before performing any task the AI must activate the LOGOS governance layer.

Required artifacts:

LOGOS_AI_ALIGNMENT_PROTOCOL
LOGOS_PROMPT_ENGINEERING_RULES
LOGOS_DIALECTIC_PROCEDURE_MANUAL

These artifacts govern:

• reasoning alignment
• prompt engineering behavior
• architectural interpretation

If any conflict exists between session context and these artifacts:

governance artifacts take precedence.
