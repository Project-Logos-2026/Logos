# VS_Code_Prompt_Contract

**System:** ARCHON_PRIME  
**Platform:** GPT → VS Code  
**Artifact Type:** Interface Specification / Prompt Contract  
**Status:** Release_Candidate_v2  
**Version:** 2.0.0  

## Required Elements

A VS Code prompt must include:
- objective
- background
- source artifacts
- target files or target surfaces
- constraints
- execution steps
- output artifacts
- validation
- logging / reporting
- rollback or correction path when mutation is involved

## Repo Reference Requirement

If the workstream is governed by repo documents, those documents must appear in `SOURCE ARTIFACTS`.

## Delivery Rule

Prompts must be delivered as one code block only.
