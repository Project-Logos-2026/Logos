## Audit Workflow

1. Define scope.
2. Identify governing artifacts.
3. Read sources without collapsing distinctions.
4. Separate findings from inference.
5. Map omissions, contradictions, and unresolved dependencies.
6. Separate required corrections from optional enhancements.
7. Report completion status.

## Mandatory Separation

Every audit must preserve:
- conceptual completeness
- specification completeness
- integration readiness
- implementation priority

## Evidence Classes

### Confirmed Finding
Directly supported by source text.

### Inference
Reasoned conclusion supported by source text but not explicitly stated.

### Recommendation
A proposed correction or enhancement based on findings and inferences.
