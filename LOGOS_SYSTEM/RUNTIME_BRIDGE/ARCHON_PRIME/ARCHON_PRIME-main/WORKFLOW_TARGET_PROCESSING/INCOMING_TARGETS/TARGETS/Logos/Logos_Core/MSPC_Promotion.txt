SECTION 0 — OBJECTIVE

Promote MSPC (Multi-Signal Process Compiler) to the FIRST runtime surface
touched by a P1 input while preserving all existing runtime invariants.

The migration must:

• NOT redesign the runtime architecture
• ONLY change ingress sequencing and artifact generation consolidation
• Preserve the EA boundary rule (only EAs enter agents/protocols)
• Preserve MTP as the final enriched user-output layer
• Preserve RGE as NON-REASONING infrastructure
• Remain fully deterministic and fail-closed

All work must be preceded by a complete repository audit.

No movement of code may occur until all audits pass.

---------------------------------------------------------------------

SECTION 1 — UNCHANGED RUNTIME INVARIANTS

The following architectural truths MUST remain unchanged:

1. Only EA artifacts may enter agents or protocols.
2. SMP and AA are internal artifact layers and never enter protocols directly.
3. RGE is non-reasoning and must never be modeled or wired as a reasoning engine.
4. MTP remains the final user-facing enrichment layer before output.
5. Logos Core retains full orchestration authority.
6. All runtime operations remain deterministic and fail-closed.

If any migration step violates these invariants the operation must halt.
