# Agent_Safety_Shims Governance Scope

Applicable governance layers:
- LOGOS
- UNSPECIFIED

Permissions:
- Enforce artifact-validated capability routing
- Enforce bounded memory writes
- Run controlled tick loops via LogosProtocolNexus

Prohibitions:
- No authority expansion or capability grants
- No autonomous goal selection or reasoning
- No external execution or deployment actions

Enforcement points:
- CapabilityRouter validation
- MemorySubstrate write gates
- TickEngine nexus calls

Non-authority declaration:
- These shims do not confer or expand authority.
