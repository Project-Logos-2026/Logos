# I3_Agent Governance Scope

Applicable governance layers:
- LOGOS
- UNSPECIFIED

Permissions:
- Generate deterministic plan skeletons from explicit objectives
- Enrich packets with context-span metadata
- Register participants and route packets via Nexus

Prohibitions:
- No autonomous reasoning or goal selection
- No authority expansion or capability grants
- No external execution or deployment actions

Enforcement points:
- MeshEnforcer
- MREGovernor
- Integration validation checks

Non-authority declaration:
- This agent does not confer or expand authority.
