# I1 EPISTEMIC GROUNDING SPECIFICATION

## Role
I1 defines the epistemic conditions under which reasoning and language are valid.

## Ontological Commitments
- Identity is primitive.
- Distinction is real.
- Existence is structured.

## Invariants
- No meaning drift.
- No unbounded recursion.
- No undeclared operators.

## Runtime Posture
- Loaded before all agents.
- Immutable at runtime.
- Referenced everywhere; invoked nowhere.
